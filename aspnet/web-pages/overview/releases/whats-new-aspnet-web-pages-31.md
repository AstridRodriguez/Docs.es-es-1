Este tema describe lo que es una novedad de ASP.NET Web Pages 3.1.

Esta versión corrige los errores y no presenta características nuevas.

<a id="download"></a>
## <a name="download"></a>Descargar

Las características en tiempo de ejecución se publican como paquetes de NuGet en la Galería de NuGet. Siguen todos los paquetes en tiempo de ejecución el [control de versiones semántico](http://semver.org/) especificación. El paquete más reciente de RTM de ASP.NET Web Pages 3.1 tiene la siguiente versión: "3.1.2". Puede instalar o actualizar estos paquetes a través de [NuGet](http://www.nuget.org/packages/Microsoft.AspNet.WebPages/). La versión también incluye paquetes localizados correspondientes en NuGet.
