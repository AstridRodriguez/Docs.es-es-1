====================
por [Microsoft](https://github.com/microsoft)

Este tema describe cuáles son las novedades para ASP.NET Web Pages 3.1.

Esta versión corrige los errores y no presenta las nuevas características.

<a id="download"></a>
## <a name="download"></a>Descargar

Las características en tiempo de ejecución se publican como paquetes de NuGet en la Galería de NuGet. Siguen todos los paquetes en tiempo de ejecución el [Versionamiento semántico](http://semver.org/) especificación. El paquete más reciente de ASP.NET Web Pages 3.1 RTM tiene la siguiente versión: "3.1.2". Puede instalar o actualizar estos paquetes a través de [NuGet](http://www.nuget.org/packages/Microsoft.AspNet.WebPages/). La versión también incluye los correspondientes paquetes localizados en NuGet.
