<a name="how-do-i-use-httpverbs-attributes-in-an-mvc-application"></a>Cómo: utilizar HttpVerbs atributos en una aplicación MVC?
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo Chris Pels muestra cómo utilizar los atributos HttpVerbs para controlar el acceso a las acciones de MVC. En primer lugar, se crea una aplicación de ejemplo con un controlador predeterminado y la vista de edición de la información. A continuación, se agrega una segunda acción de índice para el controlador que tiene un atributo HttpPost que lo restringe para que se llama solo cuando se usa una solicitud HTTP POST. Como un seguimiento, el atributo AcceptVerbs() se implementa como una sintaxis alternativa para Visual Studio 2008. A continuación, se analiza un uso de la HttpVerbs para evitar el riesgo de seguridad asociado al uso de una solicitud HTTP GET para realizar una eliminación de un vínculo.
