<a name="how-do-i-use-the-aspnet-ajax-updateprogress-control"></a>[¿Cómo I:] ¿Usar el Control ASP.NET AJAX UpdateProgress?
====================
por [Joe Stagner](https://github.com/JoeStagner)

En una aplicación AJAX suele haber una pausa notable entre el momento en que el explorador realiza una solicitud de red y el momento en que el explorador recibe una respuesta. Para que el usuario sepa que la conversación de red está teniendo lugar, el UpdateProgress de AJAX de ASP.NET control proporciona una respuesta inmediata mostrando una imagen animada, 'cargar' texto u otro HTML que desaparece una vez que se recibe la respuesta de la red.
