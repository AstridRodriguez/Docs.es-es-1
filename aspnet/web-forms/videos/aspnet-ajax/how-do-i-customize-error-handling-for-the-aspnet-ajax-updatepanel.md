<a name="how-do-i-customize-error-handling-for-the-aspnet-ajax-updatepanel"></a>¿Cómo personalizar I: control de errores de ASP.NET AJAX UpdatePanel
====================
por [Rob Windsor](https://twitter.com/robwindsor)

Cuando se produce un error durante el procesamiento de un postback desencadenado por un UpdatePanel de AJAX de ASP.NET, PageRequestManager de cliente mostrará una alerta (.NET 2.0 ó 3.0) o simplemente volver a producir la excepción (.NET 3.5 o posterior). En cualquier caso, la experiencia del usuario final es menor óptimo. En este vídeo, Rob Windsor indicará cómo resolver este problema si personaliza el control de errores asociados con el control UpdatePanel
