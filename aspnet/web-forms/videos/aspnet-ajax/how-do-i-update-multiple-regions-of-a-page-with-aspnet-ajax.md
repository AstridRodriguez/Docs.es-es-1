<a name="how-do-i-update-multiple-regions-of-a-page-with-aspnet-ajax"></a>[¿Cómo I:] ¿Actualizar varias regiones de una página con ASP.NET AJAX?
====================
por [Scott Golightly](https://github.com/scottgolightly)

En este vídeo se información que el control ASP.NET AJAX UpdatePanel proporciona dos propiedades que puede usar para actualizar varias regiones de una página Web en respuesta a un único evento de cliente. La propiedad UpdateMode puede establecerse como siempre, en el que el caso del panel siempre actualizará durante una devolución de datos, o establecido en condicional, en el que caso el panel solo se actualizará en respuesta a eventos específicos. La propiedad de colección de desencadenadores permite especificar los eventos que hará que el panel actualizar. Usar conjuntamente estas propiedades permiten varios controles UpdatePanel actualizar en respuesta al mismo desencadenador de eventos.
