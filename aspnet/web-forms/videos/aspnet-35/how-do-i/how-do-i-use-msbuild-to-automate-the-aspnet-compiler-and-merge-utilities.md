<a name="how-do-i-use-msbuild-to-automate-the-aspnet-compiler-and-merge-utilities"></a>[¿Cómo I:] Usar MSBuild para automatizar el compilador de ASP.NET y utilidades de mezcla
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo se muestra cómo crear un proyecto de MSBuild que Orquesta la ejecución de aspnet Chris Pels\_compilador y aspnet\_mezcla utilidades para un sitio web ASP.NET. Primero, aprenderá los conceptos básicos del proceso de compilación de un sitio web ASP.NET y el rol de las utilidades de línea de comandos de dos. A continuación se muestran cómo crear un archivo de proyecto de MSBuild y los elementos primarios como PropertyGroup y de destino. A continuación, obtenga información acerca de los detalles de cómo ejecutar aspnet\_compilador y aspnet\_mezcla utilidades desde el proyecto de MSBuild. Por último, se describen varias posibles expansiones de los conceptos principales.
