<a name="measuring-the-business-value-of-ajax"></a>Medir el valor empresarial de AJAX
====================
por Eric Lee

AJAX de ASP.NET es conocido por su capacidad para mejorar la experiencia del usuario de las aplicaciones Web. Con Visual Studio Team System también puede medir el valor empresarial de adición de AJAX de ASP.NET para las aplicaciones Web. En concreto puede usar las capacidades de carga y las pruebas Web de Visual Studio Team System para explorar cómo afecta la diferencia de tamaño de transmisión de datos HTTP a las características de rendimiento y escalabilidad de la aplicación Web.
