<a name="how-do-i-tune-web-application-performance-with-profiling"></a>[¿Cómo I:] ¿Optimizar el rendimiento de la aplicación Web con generación de perfiles?
====================
por [Chris Menegay](https://twitter.com/CMenegay)

En este vídeo se descubra que Visual Studio Team System ofrece dos métodos de generación de perfiles de una aplicación Web: muestreo e instrumentación. Instrumentación inserta marcadores en el código subyacente para que el desarrollador pueda ver detalles completos de la ejecución de la aplicación, incluidos los árboles de llamadas y asignación de objetos, para ayudar a optimizar el rendimiento de la aplicación.
