<a name="how-do-i-create-a-custom-provider-for-logging-health-monitoring-events"></a>[¿Cómo I:] Crear un proveedor personalizado para eventos de supervisión de estado de registro
====================
por [Chris Pels](https://twitter.com/chrispels)

Chris Pels muestra cómo crear un proveedor personalizado para el estado ASP.NET sistema, que registra eventos en un archivo de registro basado en texto de supervisión de estado de supervisión. Obtenga información acerca de cómo crear la clase de proveedor personalizado, que hereda de la clase BufferedWebEventProvider, comprender los métodos y propiedades que se usan para crear el proveedor y crear un sitio web que lo utiliza para escribir archivos de registro.
