<a name="how-do-i-utilize-the-css-friendly-control-adapters"></a>[¿Cómo I:] ¿Utilizar los adaptadores de Control descriptivo de CSS?
====================
por [Chris Pels](https://twitter.com/chrispels)

Los adaptadores de Control descriptivo CSS le permite modificar el código HTML que representa un control de servidor ASP.NET 2.0 estándar. En este vídeo se información que el control de menú de ASP.NET estándar representa un elemento HTML `<table>` elemento, pero con los adaptadores de Control descriptivo CSS del mismo control representará un elemento HTML `<ul>` elemento de lista, mejorar la accesibilidad y la presentación del menú.
