<a name="how-do-i-use-the-ftpwebrequest-class-to-create-an-ftp-connection-to-a-ftp-server"></a>[¿Cómo I:] Utilice la clase FtpWebRequest para crear una conexión FTP a un servidor FTP
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo Chris Pels muestra cómo utilizar la clase FtpWebRequest para crear una conexión FTP y transferir un archivo. En primer lugar, cree un formulario simple .aspx para seleccionar un archivo que se transferirá al servidor FTP. A continuación se muestran los métodos esenciales y las propiedades de la clase FtpWebRequest que deben configurarse con el fin de establecer una conexión FTP. A continuación, vea cómo obtener el contenido del archivo local para que el archivo se puede transferir mediante StreamReader y StreamWriter para escribir en el objeto FtpWebRequest. Por último, se obtiene un FtpWebResponse para determinar el resultado de la transferencia. Este enfoque puede utilizarse para transferir los archivos para una aplicación web mediante programación.
