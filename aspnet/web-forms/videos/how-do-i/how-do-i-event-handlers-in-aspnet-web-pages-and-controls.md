<a name="how-do-i-event-handlers-in-aspnet-web-pages-and-controls"></a>[¿Cómo I:] Controladores de eventos en las páginas Web ASP.NET y controles
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo Chris Pels mostrará cómo crear controladores de eventos para las páginas web ASP.NET y controles de servidor web. En primer lugar, obtenga información acerca de cómo crear página de eventos para los eventos de página estándar proporcionados por .NET Framework. Vea cómo enlazar automáticamente los eventos a una página mediante el atributo AutoEventWireup, así como enlazar mediante programación los eventos. A continuación, obtenga información acerca de cómo crear y enlazar controladores de eventos para los eventos de control de servidor web estándar. También aprenderá a conectar varios eventos para controles diferentes con el mismo controlador de eventos. Por último, vea el uso de la propiedad AutoPostbackTrue para controles de servidor web.
