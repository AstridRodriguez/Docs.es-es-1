<a name="how-do-i-export-data-to-a-comma-delimited-csv-file-for-an-application-like-excel"></a>[¿Cómo I:] Exportar datos a un archivo de (CSV) delimitado por comas para una aplicación como Excel.
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo Chris Pels muestra cómo tomar datos de una base de datos u otro origen y exportarlo a un archivo delimitado por comas que puede usarse en una aplicación como Excel. En primer lugar, se crea un conjunto de datos como un objeto DataTable. A continuación, se borra la respuesta de la solicitud de página web actual y el encabezado y el tipo de contenido se configuran como un archivo csv. A continuación, los datos reales se agregan a la secuencia de respuesta mediante la primera escritura de los encabezados de columna para el archivo csv seguido por los valores de datos. Este enfoque puede ser útil cuando los usuarios necesitan una exportación de datos, por lo que puede manipularse localmente en un programa como Excel.
