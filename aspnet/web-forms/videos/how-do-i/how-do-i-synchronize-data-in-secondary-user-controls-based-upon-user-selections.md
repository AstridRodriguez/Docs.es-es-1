<a name="how-do-i-synchronize-data-in-secondary-user-controls-based-upon-user-selections"></a>[¿Cómo puedo]: sincronizar los datos en controles de usuario secundario según las selecciones del usuario
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo Chris Pels muestra cómo crear dos controles DropDownList donde al seleccionar un elemento en la primera lista rellena los elementos en la segunda lista. En primer lugar, se crea una página ASP.NET con dos controles DropDownList, uno para el cliente y el otro para los Estados en el que opera el cliente. La lista de clientes, a continuación, se rellena en el código subyacente. A continuación, en el evento SelectedIndexChanged eventos de la lista de clientes, que los elementos de la lista de estados se rellenan según la selección. Por último, hay una explicación de cómo se puede aplicar este enfoque a otros tipos de controles y los datos.
