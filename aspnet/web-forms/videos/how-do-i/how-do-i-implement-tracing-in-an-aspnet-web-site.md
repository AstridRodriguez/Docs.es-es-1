<a name="how-do-i--implement-tracing-in-an-aspnet-web-site"></a>[¿Cómo I:]  ¿Implementar el seguimiento en un sitio Web ASP.NET?
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo Chris Pels le mostrará cómo implementar el seguimiento en un sitio web ASP.NET para supervisar o medir el rendimiento y diagnosticar errores. Obtenga información acerca de cómo agregar seguimiento a todo un sitio web, así como páginas individuales. Consulte cómo agregar mensajes de seguimiento personalizado en páginas y objetos comerciales como ayuda para evaluar el rendimiento del sitio y diagnosticar errores. Obtenga información acerca de las diferencias entre el seguimiento en una página ASP.NET y en mediante System.Diagnostics.Trace para obtener información de seguimiento en objetos comerciales. Además, se comentan escribir información de seguimiento de archivos de registro y ver la información de seguimiento en el Visor de seguimiento.
