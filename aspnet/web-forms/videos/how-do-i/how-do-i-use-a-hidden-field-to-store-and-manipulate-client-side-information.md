<a name="how-do-i-use-a-hidden-field-to-store-and-manipulate-client-side-information"></a>[¿Cómo I:] Usar un campo oculto para almacenar y manipular la información de cliente
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo Chris Pels mostrará cómo utilizar un campo oculto como una forma de administración de estado simple en una página ASP.NET y para comunicación de cliente información del lado del procesamiento. En primer lugar, consulte cómo agregar un campo oculto a una página con un control de servidor ASP.NET HiddenField y recuperar el valor del campo oculto en el código de servidor después de una devolución de datos. A continuación, obtenga información acerca de cómo manipular el valor del campo oculto en el script de cliente y, a continuación, recuperar el valor modificado en el código de servidor. Por último, ver un ejemplo de cómo usar varios campos ocultos para almacenar información de procesamiento del lado cliente que se puede registrar para procesamiento del servidor.
