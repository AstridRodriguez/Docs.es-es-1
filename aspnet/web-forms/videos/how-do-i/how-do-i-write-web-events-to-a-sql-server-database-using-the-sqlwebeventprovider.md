<a name="how-do-i-write-web-events-to-a-sql-server-database-using-the-sqlwebeventprovider"></a>[¿Cómo I:] Escribir eventos Web en una base de datos SQL Server mediante el SqlWebEventProvider
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo Chris Pels muestra cómo usar el SqlWebEventProvider de supervisión de estado ASP.NET para registrar los errores en un sitio web a una base de datos de SQL Server. En primer lugar, obtenga información acerca de la función de los eventos de supervisión de estado ASP.NET y el proveedor. A continuación se muestran cómo configurar una base de datos de SQL Server con los objetos necesarios para los eventos mediante aspnet de supervisión de estado de grabación\_regiis utilidad, la misma utilidad que se usa para configurar la pertenencia de ASP.NET. A continuación, aprenda a configurar en el archivo web.config para registrar eventos en un sitio web ASP.NET a la base de datos de SQL Server recién creada de supervisión de estado. Como parte de esta configuración, vea cómo el archivo web.config raíz en .NET Framework 2.0 tiene proveedores y eventos definidos que se pueden aprovechar al configurar la supervisión de estado. Estos aspectos básicos son una base en la que se pueden crear eventos personalizados que registrar su propia información específica en una base de datos de SQL Server en un sitio web ASP.NET.
