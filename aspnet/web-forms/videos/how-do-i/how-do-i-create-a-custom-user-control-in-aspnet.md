<a name="how-do-i--create-a-custom-user-control-in-aspnet"></a>[¿Cómo I:]  ¿Crear un Control de usuario personalizado en ASP.NET?
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo, Chris Pels indicará cómo crear un control de usuario personalizado de ASP.NET que se puede usar en todo un sitio web o a través de muchos sitios web. Iniciar alejar al aprender los conceptos básicos de creación de un control de usuario que se usará en varias páginas, incluida la validación, controlar el formato utilizando Aplicar estilo a los elementos y cargar o guardar datos del usuario elementos de la interfaz. A lo largo de las consideraciones de vídeo para la creación de usuario se describen los controles que se pueden mantener fácilmente y usar en varias páginas.
