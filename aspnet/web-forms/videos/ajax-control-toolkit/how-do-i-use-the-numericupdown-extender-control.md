<a name="how-do-i-use-the-numericupdown-extender-control"></a>[¿Cómo I:] ¿Usar el Control del dispositivo Extender NumericUpDown?
====================
por [Joe Stagner](https://github.com/JoeStagner)

En este vídeo se examina el control de extender NumericUpDown desde el Kit de herramientas de Control de AJAX de ASP.NET. Veremos cómo agrega el dispositivo extender NumericUpDown "Subir" y "Bajar" botones a un control de cuadro de texto de ASP.NET estándar. Los botones permiten al usuario incrementar o disminuir un número, desplazarse por una lista de valores conocidos o llamar a un servicio Web para recuperar el valor siguiente. De esta forma el programador puede controlar totalmente los valores que el usuario puede enviar a una aplicación Web.
