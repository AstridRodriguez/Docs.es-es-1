<a name="how-do-i-use-the-aspnet-ajax-maskededit-controls"></a>[¿Cómo I:] ¿Use los controles ASP.NET AJAX MaskedEdit?
====================
por [Joe Stagner](https://github.com/JoeStagner)

El Kit de herramientas de Control de AJAX de ASP.NET proporciona el control MaskedEdit y el control MaskedEditValidator que trabajan juntos para solicitar, estilo y validar los datos que el usuario escribe en un cuadro de texto. Esto permite que la página Web para proporcionar al usuario información visual inmediata sobre si los datos introducidos son de un tipo válido y el formato.
