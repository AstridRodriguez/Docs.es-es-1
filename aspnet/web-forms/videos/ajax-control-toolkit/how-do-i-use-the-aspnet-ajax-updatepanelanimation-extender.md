<a name="how-do-i-use-the-aspnet-ajax-updatepanelanimation-extender"></a>[¿Cómo I:] ¿Usar el dispositivo Extender UpdatePanelAnimation de AJAX de ASP.NET?
====================
por [Joe Stagner](https://github.com/JoeStagner)

Al agregar animación a una actualización parcial de página que se puede informar al usuario cuando una actualización está produciendo y cuándo ha finalizado. El extensor UpdatePanelAnimation agrega efectos de animación para el control de ASP.NET AJAX UpdatePanel, haciendo uso de la plataforma de animación enriquecido del núcleo de ASP.NET AJAX Extensions. Este vídeo muestra cómo puede definir la secuencia de efectos de la UpdatePanelAnimation y cómo puede ajustar la animación mediante el uso de las definiciones de ScriptAction.
