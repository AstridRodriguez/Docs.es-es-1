<a name="how-do-i-use-the-aspnet-ajax-mutuallyexclusive-checkbox-extender"></a>[¿Cómo I:] ¿Usar el extensor de casilla de verificación MutuallyExclusive ASP.NET AJAX?
====================
por [Joe Stagner](https://github.com/JoeStagner)

El control de MutuallyExclusive desde el Kit de herramientas de Control de AJAX de ASP.NET permite al desarrollador configurar un número de casillas de verificación que se excluyen mutuamente, lo que garantiza que el usuario sólo puede comprobar una única opción. Mientras que una lista de botones de radio de ofrece un comportamiento similar, la necesidad de una casilla de verificación para hacer clic por el usuario, el control de MutuallyExclusive ideal para recibir la confirmación del usuario, como Aceptar o no acepta los términos de uso.
