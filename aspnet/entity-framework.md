<a name="entity-framework"></a>Entity Framework
====================
Entity Framework (EF) es un asignador relacional de objetos que permite a los desarrolladores de .NET trabajar con datos relacionales con objetos específicos del dominio. Se elimina la necesidad de la mayoría del código de acceso a datos que normalmente los desarrolladores deben escribir.


- [Obtenerlo:](https://msdn.com/data/ee712906) agregarla a su proyecto y empezar a aprovechar este eficaz O/RM.
- [Introducción:](https://msdn.com/data/ee712907) visite la guía para los vídeos y tutoriales que le ayudará a aprovechar al máximo sus datos.
- [Ayuda de Get:](https://msdn.com/data/hh913619) obtener información sobre cómo obtener ayuda de los expertos y contribuir a la Comunidad de datos acerca del desbordamiento de pila.
- [Contribuir:](https://github.com/aspnet/EntityFramework6) averiguar cómo pueden ayudar a mejorar Entity Framework, un modelo de desarrollo abierto, incluso, visite el sitio de GitHub.
