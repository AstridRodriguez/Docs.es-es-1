### <a name="partial-view-discovery"></a><span data-ttu-id="1305f-134">Detección de la vista parcial</span><span class="sxs-lookup"><span data-stu-id="1305f-134">Partial View Discovery</span></span>

```text

// If none is found, searches the Shared folder

// A view with this name must be in the same folder
```
