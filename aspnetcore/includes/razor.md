En este tutorial se muestra ASP.NET Core MVC con controladores y vistas. Las páginas de Razor son una nueva alternativa en ASP.NET Core 2.0, un modelo de programación basado en páginas que facilita la compilación de interfaces de usuario web y hace que sean más productivas. Se recomienda probar el tutorial de las [páginas de Razor](xref:tutorials/razor-pages/razor-pages-start) antes que la versión MVC. El tutorial de las páginas de Razor:

* Es el método preferido para el desarrollo de nuevas aplicaciones.
* Es más fácil de seguir.
* Abarca más características.
