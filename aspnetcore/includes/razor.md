En este tutorial se muestra ASP.NET Core MVC con controladores y vistas. Las páginas de Razor son una nueva alternativa en ASP.NET Core 2.0, un modelo de programación basado en páginas que facilita la compilación de interfaces de usuario web y hace que sean más productivas. Se recomienda probar el tutorial de las páginas de Razor antes que la versión MVC. Para obtener una versión de las páginas de Razor para este tutorial, consulte las [páginas de Razor](xref:mvc/razor-pages/index).
