# <a name="primitives-in-aspnet-core"></a>Primitivas en ASP.NET Core

Las primitivas de ASP.NET Core son los bloques de creación de bajo nivel compartidos por extensiones de la plataforma. Puede usar estos bloques de creación en su propio código.
