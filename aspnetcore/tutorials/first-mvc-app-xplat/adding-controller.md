# <a name="adding-a-controller-to-an-aspnet-core-mvc-app-with-visual-studio-code"></a>Agregar un controlador a una aplicación de la MVC de ASP.NET Core con Visual Studio Code

Por [Rick Anderson](https://twitter.com/RickAndMSFT)

[!INCLUDE[adding-controller](../../includes/mvc-intro/adding-controller1.md)]

* En **VS Code**, haga clic en el icono **EXPLORER** y luego presione CTRL y haga clic (derecho) en **Controladores > Nuevo archivo** y asigne el nombre HelloWorldController.cs al nuevo archivo.
